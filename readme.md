# Node.js Real-time chat Back-end
Is a server for a little Real-time chat app (Is just for learning)

# I'm using
  - Nodejs
  - Socket.io

### Installation
Install the dependencies and devDependencies and start the server.

```sh
$ npm install
$ npm run dev
```
