# Node.js Real-time chat Back-end
Server for a little Real-time chat app (It's just for learning)

# Relevant libs that i'm using
  - Nodejs
  - Socket.io

### Installation
Install the dependencies and devDependencies, so start the server.

```sh
$ npm install
$ npm run dev
```
